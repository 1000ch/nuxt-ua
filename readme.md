# nuxt-ua

UserAgent detector using [Platform.js](https://github.com/bestiejs/platform.js) for Nuxt.

## Install

```bash
$ npm install --save nuxt-ua
```

## Usage

Configure `nuxt.config.js`.

```js
module.exports = {
  modules: [
    'nuxt-ua'
  ]
}
```
